# MERN Todo App

A full-stack Todo Management Application built using the MERN stack (MongoDB, Express.js, React.js, and Node.js). The application allows users to manage daily tasks efficiently with secure authentication and real-time updates.

## Features

- User Registration & Login
- JWT Authentication
- Create Todo
- Update Todo
- Delete Todo
- Mark Tasks as Completed
- Responsive UI
- Secure Backend APIs

## Tech Stack

### Frontend
- React.js
- React Router
- Axios
- CSS

### Backend
- Node.js
- Express.js
- MongoDB
- Mongoose
- JWT
- bcrypt
